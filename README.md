# Private-Data-Vue
Safely store and manage your private data just on the website.



# Directory Structure
```text
├── publish/
└── src/
    ├── api/                       // api 接口目录
    ├── assets/                    // 静态资源目录
    ├── common/                    // 公共组件目录
    ├── components/                // 页面组件目录
    ├── model/                    // 实体类库目录
    ├── plugin/                   // 插件类库目录
    ├── router/                    // 路由配置目录
    ├── store/                     // 状态管理目录
    ├── style/                    // 公共样式目录
    ├── util/                     // 工具类库目录
    ├── view/                     // 页面组件目录
    ├── App.vue
    ├── main.ts
    ├── env.d.ts
├── tests/                         // 单元测试目录
├── index.html
├── tsconfig.json                  // TypeScript 配置文件
├── vite.config.ts                 // Vite 配置文件
└── package.json
```
